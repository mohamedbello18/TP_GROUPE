# TP_GROUPE - Team Portfolio

A collaborative portfolio website for our team members to showcase skills, projects, and achievements.

## Project Structure

- `index.html` - Main HTML file containing the portfolio structure
- `styles.css` - CSS styles for the portfolio
- `README.md` - Project documentation

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser to view the portfolio

## How to Contribute

Team members can contribute by:

1. **Adding your profile**: Edit the team section in `index.html` to add your team card
2. **Adding projects**: Add new project cards in the projects section
3. **Improving styles**: Enhance the CSS in `styles.css`

### Steps to Contribute

1. Create a new branch for your changes
2. Make your modifications
3. Submit a pull request for review
